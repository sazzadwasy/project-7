Ques: How does javaScript works ?

Ans: Everything in javaScript happens inside an execution context,javaScript is a syncronous single threated language.In execution context there are two component in it.The first is memory component.This is the place where all the variables and functions stored as key-value pairs. This component is also known as variable environment.Second component in execution context is code component,this is the place where code is executed one line at a time.This is also known as thread of execution javaScript can only execute one command at a time in a specific order.It can only go to the next line once the current line has been finished executing.

Ques: what is the difference between localStorage and sessionStorage ?

Ans: Web Storage Api is used by developers to store some data in the web browser.This data is nothing but key-value pair of strings.There are two machanism to store this data.One machanism is sessionStorage and the second machanism is localStorage.When data is stored in sessionStorage, that mean the data is store there for a particular session.Foe Example if a user visit the website the session is started and the sessionStorage data will remain till user is in the website.The localStorage is also the sessionStorage but the advantage in localStorage is, data is not removed by itself.If the user closes the browser still that data stored in the localStorage is not removed.If we talk about the memory capasity localStorage has the highest memory capasity compare to the sessionStorage.
